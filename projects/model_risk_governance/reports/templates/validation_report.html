1. Executive Summary

This report documents the independent validation of the {{ model_name }}, a Logistic Regression classifier trained to predict the probability of default for personal loans at origination. Training data covers Lending Club loans issued prior to {{ train_period }}; model performance is evaluated on a held-out test set and monitored on loans issued from {{ monitor_period }}.

The model was developed in accordance with the SR 11-7 model lifecycle framework and covers data validation, leakage controls, model development, threshold governance, drift monitoring, calibration assessment, and fair lending analysis.
