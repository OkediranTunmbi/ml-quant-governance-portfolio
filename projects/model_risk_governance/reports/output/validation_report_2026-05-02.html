1. Executive Summary

This report documents the independent validation of the Logistic Regression Credit Default Model, a Logistic Regression classifier trained to predict the probability of default for personal loans at origination. Training data covers Lending Club loans issued prior to < 2015-01-01; model performance is evaluated on a held-out test set and monitored on loans issued from >= 2015-01-01.

The model was developed in accordance with the SR 11-7 model lifecycle framework and covers data validation, leakage controls, model development, threshold governance, drift monitoring, calibration assessment, and fair lending analysis.

Overall status: AUC-ROC (test) = 0.6962, Gini = 0.3925, KS = 0.2844. Performance on monitor: AUC = 0.7142 (OK).
